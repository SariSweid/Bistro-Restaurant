📌 Project Overview

Bistro is a desktop-based restaurant reservation management system built in Java.
It allows customers and staff to efficiently manage table reservations, reduce waiting times, and streamline restaurant operations.

🚀 Features
Table reservation system
Waiting list management
Reservation cancellation
Membership system for customers

🔐 Role-based access:
Customer
Member
Supervisor
Manager

📊 Real-time table availability tracking

🛠️ Tech Stack
Java
JavaFX
MySQL
JDBC
